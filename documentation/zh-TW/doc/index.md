#### 目錄


* 快速使用
	- 安裝
		1. 用iso光碟安裝Stem OS.
			- 首先先從 [Download](http://sourceforge.net/projects/stemos/files/latest/download) 頁面下載iso檔.
			- 透過光碟燒錄軟體把iso檔燒到CD/DVD光碟片中.
			- 找一台x86的電腦並用光碟片安裝Stem OS.
		2. 執行 Stem 應用程式.
			- 把 Stem 應用程式放到USB隨身碟的根目錄(/).
			- 把USB隨身碟plug in到有安裝Stem OS的x86電腦中.
